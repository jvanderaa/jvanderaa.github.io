Network Automation Guides

Years of writing blog posts on network automation led me to a simple realization: engineers need more than scattered articles. They need a structured, hands-on path that goes from zero to working automation - with real devices, real output, and no fluff.

These guides are lab-driven references built for network engineers ready to automate. Each one covers a core tool - Ansible, Netmiko, NAPALM, Nornir, or Go - using Arista EOS in Containerlab so you can follow along and run every example yourself.
